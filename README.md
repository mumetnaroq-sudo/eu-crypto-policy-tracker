# EU Crypto Policy Tracker

Real-time EU cryptocurrency policy intelligence for prediction markets.

## Features

- **27 EU Countries**: Comprehensive policy tracking across all EU jurisdictions
- **Live Probability Scores**: AI-generated predictions for regulatory outcomes
- **Key Actor Analysis**: Politician profiles with influence ratings
- **Intelligence Alerts**: Real-time notifications for policy changes
- **Terminal Aesthetic**: Distinctive green-on-black interface

## Tech Stack

- Next.js 14
- TypeScript
- Static Export (for Vercel deployment)

## Development

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run build
# Deploy dist/ folder to Vercel
```

## Data Sources

- ESMA (European Securities and Markets Authority)
- ECB (European Central Bank)
- EU Parliament legislative tracking
- National regulatory bodies

## Disclaimer

This tool provides intelligence signals for informational purposes only. Not financial advice. Predictions reflect AI analysis and should not be the sole basis for trading decisions.
